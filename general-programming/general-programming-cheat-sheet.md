# General Programming

## Logical Expressions
* if statements
* else and elseif statements
* switch statements (usually used with evaluating the value of strings):
```
  switch (value) {
    case 'case1':
      statement
      break;
    case 'case2':
      statement
      break;
    default:
      statement
      break;
    }
```

## Operators
### Comparison Operators
* equal: ==
* identical: === (same type and value)
* compare: > < >= <= <>
* not equal: !=
* not identical: !==

### Logical Operators
* and: && (both parts must be true)
* or: ||
* not: !
